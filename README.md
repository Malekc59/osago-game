# 🚗 ОСАГО: Умный водитель

Игра про вождение с элементами ОСАГО. Чем меньше аварий — тем дешевле полис!

## 🎮 Как играть

1. Выбери город, стаж и мощность двигателя — рассчитается стоимость полиса
2. Управляй машиной: тапай левую/правую часть экрана (или стрелки на клавиатуре)
3. Уворачивайся от препятствий — каждая авария повышает КБМ и цену полиса
4. Через 60 секунд узнаешь результат!

## 💰 Монетизация

- **Баннерная реклама** — на экране формы
- **Межстраничная реклама** — после каждой игры
- **Rewarded video** — «второй шанс» после аварии
- **Лидерборд** — соревнование с друзьями

## 🚀 Запуск

Открой `index.html` в браузере или разверни через GitHub Pages.

## 📁 Структура

- `index.html` — разметка и экраны
- `script.js` — логика игры, VK Bridge, реклама
- `style.css` — стили и анимации
